## Stack

- LIFO
- Build on top of linked list data structure, actually high level API.
- Simple API: `push()`, `pop()`, `peak()`, `isEmpty()`

![stack](../images/stack.png)

- Access, search **O(n)**
- **Killer feature:** `push()`, `pop()` ***O(1)***
